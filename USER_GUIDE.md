## Getting Started

### Logging In

1. Navigate to the CandlPage URL
2. Enter your email address and password
3. Click "Sign In"
4. You'll be directed to either:
   - **Admin Dashboard** - If you're an admin user
   - **Client Portal** - If you're a client user

### First Time Setup

Admin users should:
1. Review all available tools on the dashboard
2. Set up automated campaigns if needed
3. Create client accounts for any clients who need portal access

## SEO/AEO Audit Tool

Generate comprehensive SEO and Answer Engine Optimization audits for any website.

### Running an Audit

1. **Enter Website URL**
   - Click on "SEO/AEO Audit" from the dashboard
   - Enter the full URL (e.g., `https://example.com`)
   - Click "Run Audit"

2. **Wait for Analysis**
   - The audit typically takes 30-60 seconds
   - Progress indicator shows status
   - AI analyzes technical SEO, content, and AEO factors

3. **Review Results**
   - **Overall Score** (0-100) appears at the top
   - **Score Breakdown** by category
   - **Detailed Recommendations** with priority levels
   - **Implementation Snippets** - Copy-paste code fixes

### Understanding Scores

- **90-100**: Excellent - Minor optimizations only
- **70-89**: Good - Some improvements recommended
- **50-69**: Needs Work - Multiple issues to address
- **Below 50**: Critical - Significant problems detected

### Categories Analyzed

- **Technical SEO**: Site speed, mobile-friendliness, crawlability
- **On-Page SEO**: Meta tags, headings, content structure
- **Content Quality**: Keyword usage, readability, value
- **Answer Engine Optimization**: Featured snippet potential, schema markup
- **User Experience**: Navigation, accessibility, engagement

### Exporting Results

1. Review the audit results
2. Click "Export to CSV" button
3. Save the file for client delivery or records
4. CSV includes all recommendations and priorities

### Assigning to Clients

1. After running an audit, click "Assign to Client"
2. Select the client from the dropdown
3. Click "Assign"
4. Client can now view this audit in their portal

## Security Scanner

Perform comprehensive cybersecurity vulnerability assessments.

### Running a Security Scan

1. **Enter Target Information**
   - Click "Security Scanner" from dashboard
   - Enter domain or IP address
   - Optionally add specific scan parameters
   - Click "Run Security Scan"

2. **Analysis Process**
   - Scan takes 1-3 minutes depending on target size
   - Checks for common vulnerabilities
   - Cross-references CVE database
   - Generates risk assessment

3. **Review Findings**
   - **Risk Level** - Overall security grade
   - **Vulnerabilities Found** - Categorized by severity
   - **CVE References** - Linked to official databases
   - **Remediation Steps** - Specific fix instructions

### Severity Levels

- **Critical**: Immediate action required
- **High**: Address as soon as possible
- **Medium**: Plan for resolution
- **Low**: Monitor and address when convenient
- **Informational**: Best practice recommendations

### Understanding Results

Each vulnerability includes:
- **Description**: What the issue is
- **Impact**: Potential consequences
- **Recommendation**: How to fix it
- **Code Snippets**: Implementation examples
- **Resources**: Links to documentation

### Exporting Security Reports

1. Review scan results
2. Click "Export Report"
3. Choose format (CSV or detailed report)
4. Save for client delivery or compliance records

### Assigning Security Scans to Clients

1. Click "Assign to Client" after scan completes
2. Select client from dropdown
3. Client receives read-only access to the report

## Content Suite

Generate AI-powered content for various marketing needs.

### Keyword Research

1. **Enter Parameters**
   - Industry/niche
   - Geography (optional)
   - Target audience (optional)
   - Click "Generate Keywords"

2. **Review Results**
   - 20-30 keyword suggestions
   - Search volume estimates
   - Difficulty scores (1-100)
   - Search intent classification
   - Competitive analysis

3. **Export Keywords**
   - Click "Export to CSV"
   - Import into SEO tools or content calendar

### Press Release Generation

1. **Fill in Details**
   - Company name
   - Announcement details
   - Style preference (Professional, Casual, Technical)
   - Additional context (optional)
   - Click "Generate Press Release"

2. **Review & Edit**
   - Complete press release appears
   - Includes: Headline, dateline, lead, body, boilerplate
   - Copy and paste into your system
   - Edit as needed for brand voice

3. **Export**
   - Click "Copy to Clipboard" or "Export"
   - Ready for distribution

### SEO Article Generation

1. **Set Parameters**
   - Topic/title
   - Target keywords (comma-separated)
   - Tone (Professional, Casual, Technical, Friendly)
   - Length (500-3000 words)
   - Click "Generate Article"

2. **Content Delivery**
   - Full article with optimized structure
   - H2/H3 headings properly formatted
   - Meta description included
   - Keywords naturally integrated
   - Ready for CMS upload

3. **Edit & Refine**
   - Review for accuracy
   - Adjust tone if needed
   - Add brand-specific details
   - Export or copy content

### Best Practices

- Be specific with prompts for better results
- Review all AI content before publishing
- Customize generated content for brand voice
- Verify facts and statistics
- Add unique insights and examples

## Lead Generator

Discover high-quality business leads with detailed opportunity scoring.

### Finding Leads

1. **Enter Search Criteria**
   - Industry/vertical
   - Geographic location (optional)
   - Company size (optional)
   - Other filters
   - Click "Find Leads"

2. **Analysis Process**
   - AI searches for businesses matching criteria
   - Analyzes websites for opportunity indicators
   - Scores SEO performance
   - Identifies improvement opportunities
   - Takes 2-5 minutes per batch

3. **Review Lead List**
   - Company name and website
   - SEO score (0-100)
   - Opportunity score
   - Key findings
   - Contact potential

### Understanding Lead Scores

**SEO Score Components:**
- Technical SEO health
- Content quality
- Mobile optimization
- Page speed
- Backlink profile

**Opportunity Score Factors:**
- Revenue potential
- SEO improvement potential
- Competition level
- Market position
- Quick win opportunities

### Lead Details

Click on any lead to see:
- Full SEO audit summary
- Specific issues found
- Improvement recommendations
- Estimated impact
- Talking points for outreach

### Exporting Leads

1. Select leads you want to export (or export all)

   - Contact information
   - Scores and metrics
   - Key talking points
   - Next steps

### CRM Integration

- Export leads to CSV

- Use talking points for personalized outreach
- Track conversion rates

## Client Management

Create and manage client portal access.

### Creating Client Accounts

1. **Click "Client Management"**
2. **Click "Add New Client"**
3. **Enter Client Information**
   - Client company name
   - Primary contact name
   - Email address
   - Password (temporary)
   - Click "Create Client User"

4. **Confirmation**
   - Client account created
   - Login credentials displayed
   - Share credentials securely with client

### Managing Existing Clients

**View All Clients:**
- List shows all client accounts
- Company name
- Contact email
- Number of assigned reports
- Account status

**Edit Client:**
1. Click on client name
2. Update information
3. Save changes

**Reset Password:**
1. Click "Reset Password" next to client
2. New temporary password generated
3. Share securely with client

**Deactivate Client:**
1. Click "Deactivate" button
2. Client loses portal access
3. Data retained for reactivation

### Assigning Reports to Clients

**From Any Tool:**
1. Complete an audit or scan
2. Click "Assign to Client"
3. Select client from dropdown
4. Click "Assign"

**From Client Management:**
1. View client details
2. Click "Assign Reports"
3. Select from available reports
4. Click "Assign Selected"

### Client Portal Features

Clients can:
- Log in with their credentials
- View assigned audits and reports
- Download reports as CSV
- See historical reports
- Cannot create new audits

### Removing Report Access

1. Go to client details
2. Find assigned report
3. Click "Remove Access"
4. Report no longer visible to client

## Tips & Best Practices

### SEO Audits

- Run audits on full domains, not individual pages
- Schedule regular re-audits to track progress
- Use audit results as sales tools for clients
- Focus on high-priority recommendations first

### Security Scans

- Run scans during off-peak hours when possible
- Address Critical and High severity issues immediately
- Keep records of all scans for compliance
- Re-scan after implementing fixes

### Content Generation

- Always review and fact-check AI content
- Add company-specific examples and data
- Customize tone for each client's brand
- Use keyword research to guide content topics

### Lead Generation

- Be specific with search criteria for better matches
- Focus on opportunity score, not just SEO score
- Use audit findings as conversation starters
- Track conversion rates by industry/criteria

### Automated Campaigns

- Start with one campaign and optimize before scaling
- Review weekly reports for trends
- Adjust targeting based on lead quality
- Use industry analysis for thought leadership

### Client Management

- Set strong temporary passwords
- Instruct clients to change passwords on first login
- Regularly review client access and clean up inactive accounts
- Assign reports promptly for better client experience

## Troubleshooting

### Common Issues

**Login Problems**
- Verify email and password are correct
- Check for caps lock
- Clear browser cache and cookies
- Try incognito/private browsing mode

**Audit Taking Too Long**
- Some sites take longer to analyze
- Wait up to 2 minutes before retrying
- Check internet connection
- Try a different browser

**Export Not Working**
- Ensure pop-ups are not blocked
- Try a different browser
- Check download folder settings
- Verify sufficient disk space

**Client Can't Log In**
- Verify credentials were shared correctly
- Reset password if needed
- Check that account is active
- Ensure email is correct

**Reports Not Showing for Client**
- Verify report was assigned
- Check client account is active
- Try refreshing the page
- Log out and log back in

**Automated Campaign Not Running**
- Check campaign status (active/paused)
- Verify schedule settings
- Check email delivery settings
- Review campaign logs

### Performance Optimization

- Close unused browser tabs
- Clear cache periodically
- Use modern browsers (Chrome, Firefox, Safari, Edge)
- Ensure stable internet connection
- Allow sufficient time for AI analysis

### Data Export Issues

- Large exports may take time
- Check browser download settings
- Verify file permissions
- Try exporting smaller batches

### Getting Help

If issues persist:
1. Note the exact error message
2. Document steps taken
3. Take screenshots if helpful
4. Contact C&L Strategy support
