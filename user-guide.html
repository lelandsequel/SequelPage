Getting Started

Logging In

1
Navigate to the CandlPage URL
2
Enter your email address and password
3
Click "Sign In"
4
You'll be directed to either the Admin Dashboard or Client Portal
First Time Setup: Admin users should review all available tools, set up automated campaigns if needed, and create client accounts for portal access.

Admin Dashboard Overview

The admin dashboard provides access to six main tools, each displayed as a tool card with an icon, title, description, features list, and launch button.

Navigation Tips

  • Click any "Launch Tool" button to open that tool
  • Use the "Sign Out" button in the top right to log out
  • System status indicator shows operational status

SEO/AEO Audit Tool

Generate comprehensive SEO and Answer Engine Optimization audits for any website.

Running an Audit

1
Enter Website URL
Click on "SEO/AEO Audit" from the dashboard, enter the full URL (e.g., https://example.com), and click "Run Audit"
2
Wait for Analysis
The audit typically takes 30-60 seconds. Progress indicator shows status while AI analyzes technical SEO, content, and AEO factors.
3
Review Results
View overall score (0-100), score breakdown by category, detailed recommendations with priority levels, and implementation snippets.

Understanding Scores

Score Range Rating Action Needed
90-100 Excellent Minor optimizations only
70-89 Good Some improvements recommended
50-69 Needs Work Multiple issues to address
Below 50 Critical Significant problems detected

Categories Analyzed

  • Technical SEO: Site speed, mobile-friendliness, crawlability
  • On-Page SEO: Meta tags, headings, content structure
  • Content Quality: Keyword usage, readability, value
  • Answer Engine Optimization: Featured snippet potential, schema markup
  • User Experience: Navigation, accessibility, engagement

Exporting Results

  1. Review the audit results
  2. Click "Export to CSV" button
  3. Save the file for client delivery or records
  4. CSV includes all recommendations and priorities

Assigning to Clients

  1. After running an audit, click "Assign to Client"
  2. Select the client from the dropdown
  3. Click "Assign"
  4. Client can now view this audit in their portal

Security Scanner

Perform comprehensive cybersecurity vulnerability assessments.

Running a Security Scan

1
Enter Target Information
Click "Security Scanner" from dashboard, enter domain or IP address, optionally add specific scan parameters, and click "Run Security Scan"
2
Analysis Process
Scan takes 1-3 minutes depending on target size. Checks for common vulnerabilities, cross-references CVE database, and generates risk assessment.
3
Review Findings
View risk level, vulnerabilities found (categorized by severity), CVE references, and remediation steps.

Severity Levels

Severity Action Required
Critical Immediate action required
High Address as soon as possible
Medium Plan for resolution
Low Monitor and address when convenient
Informational Best practice recommendations

Understanding Results

Each vulnerability includes:

  • Description: What the issue is
  • Impact: Potential consequences
  • Recommendation: How to fix it
  • Code Snippets: Implementation examples
  • Resources: Links to documentation

Content Suite

Generate AI-powered content for various marketing needs.

Keyword Research

1
Enter Parameters
Industry/niche, geography (optional), target audience (optional), then click "Generate Keywords"
2
Review Results
20-30 keyword suggestions with search volume estimates, difficulty scores (1-100), search intent classification, and competitive analysis
3
Export Keywords
Click "Export to CSV" and import into SEO tools or content calendar

Press Release Generation

Fill in Details:

  • Company name
  • Announcement details
  • Style preference (Professional, Casual, Technical)
  • Additional context (optional)

Complete press release appears with headline, dateline, lead, body, and boilerplate. Copy and paste into your system, edit as needed for brand voice.

SEO Article Generation

1
Set Parameters
Topic/title, target keywords (comma-separated), tone (Professional, Casual, Technical, Friendly), length (500-3000 words)
2
Content Delivery
Full article with optimized structure, H2/H3 headings, meta description, and naturally integrated keywords
3
Edit & Refine
Review for accuracy, adjust tone, add brand-specific details, export or copy content
Best Practices: Be specific with prompts for better results. Always review AI content before publishing. Customize for brand voice. Verify facts and statistics. Add unique insights and examples.

Lead Generator

Discover high-quality business leads with detailed opportunity scoring.

Finding Leads

1
Enter Search Criteria
Industry/vertical, geographic location (optional), company size (optional), other filters, then click "Find Leads"
2
Analysis Process
AI searches for businesses matching criteria, analyzes websites for opportunity indicators, scores SEO performance, and identifies improvement opportunities (takes 2-5 minutes per batch)
3
Review Lead List
Company name and website, SEO score (0-100), opportunity score, key findings, contact potential

Understanding Lead Scores

SEO Score Components:

  • Technical SEO health
  • Content quality
  • Mobile optimization
  • Page speed
  • Backlink profile

Opportunity Score Factors:

  • Revenue potential
  • SEO improvement potential
  • Competition level
  • Market position
  • Quick win opportunities

Lead Details

Click on any lead to see full SEO audit summary, specific issues found, improvement recommendations, estimated impact, and talking points for outreach.

Exporting Leads

  1. Select leads you want to export (or export all)
  2. Click "Export to CSV"
  3. CSV includes contact information, scores and metrics, key talking points, and next steps

Automated Discovery

Set up automated weekly lead generation campaigns.

Creating a Campaign

1
Click "Automated Discovery"
2
Set Campaign Parameters
Campaign name, industry/vertical, geographic targeting, number of leads per week, delivery email address
3
Configure Schedule
Choose day of week for delivery, set time (optional), select report format
4
Launch Campaign
Click "Create Campaign" - system automatically runs weekly and sends email reports

Campaign Dashboard

View all active campaigns with:

  • Campaign name and status
  • Next scheduled run
  • Last run results
  • Total leads generated
  • Campaign performance

Managing Campaigns

Action Steps
Edit Campaign Click on campaign name, modify parameters, save changes
Pause Campaign Click "Pause" button, campaign stops running, click "Resume" to restart
Delete Campaign Click "Delete" button, confirm deletion, campaign permanently removed

Email Reports

Weekly emails include:

  • Executive summary
  • New leads discovered
  • Industry insights
  • Top opportunities
  • CSV attachment with full data

Client Management

Create and manage client portal access.

Creating Client Accounts

1
Click "Client Management"
2
Click "Add New Client"
3
Enter Client Information
Client company name, primary contact name, email address, password (temporary)
4
Click "Create Client User" - credentials will be displayed to share securely with client

Managing Existing Clients

View All Clients:

List shows all client accounts with company name, contact email, number of assigned reports, and account status.

Quick Actions:

  • Edit Client: Click on client name, update information, save changes
  • Reset Password: Click "Reset Password" next to client, new temporary password generated, share securely
  • Deactivate Client: Click "Deactivate" button, client loses portal access, data retained for reactivation

Assigning Reports to Clients

From Any Tool: Complete an audit or scan, click "Assign to Client", select client from dropdown, click "Assign"
From Client Management: View client details, click "Assign Reports", select from available reports, click "Assign Selected"

Client Portal Features

Clients can:

  • Log in with their credentials
  • View assigned audits and reports
  • Download reports as CSV
  • See historical reports
  • Cannot create new audits

Tips & Best Practices

SEO Audits

  • Run audits on full domains, not individual pages
  • Schedule regular re-audits to track progress
  • Use audit results as sales tools for clients
  • Focus on high-priority recommendations first

Security Scans

  • Run scans during off-peak hours when possible
  • Address Critical and High severity issues immediately
  • Keep records of all scans for compliance
  • Re-scan after implementing fixes

Content Generation

  • Always review and fact-check AI content
  • Add company-specific examples and data
  • Customize tone for each client's brand
  • Use keyword research to guide content topics

Lead Generation

  • Be specific with search criteria for better matches
  • Focus on opportunity score, not just SEO score
  • Use audit findings as conversation starters
  • Track conversion rates by industry/criteria

Automated Campaigns

  • Start with one campaign and optimize before scaling
  • Review weekly reports for trends
  • Adjust targeting based on lead quality
  • Use industry analysis for thought leadership

Client Management

  • Set strong temporary passwords
  • Instruct clients to change passwords on first login
  • Regularly review client access and clean up inactive accounts
  • Assign reports promptly for better client experience

Troubleshooting

Common Issues

Login Problems

  • Verify email and password are correct
  • Check for caps lock
  • Clear browser cache and cookies
  • Try incognito/private browsing mode

Audit Taking Too Long

  • Some sites take longer to analyze
  • Wait up to 2 minutes before retrying
  • Check internet connection
  • Try a different browser

Export Not Working

  • Ensure pop-ups are not blocked
  • Try a different browser
  • Check download folder settings
  • Verify sufficient disk space

Client Can't Log In

  • Verify credentials were shared correctly
  • Reset password if needed
  • Check that account is active
  • Ensure email is correct

Reports Not Showing for Client

  • Verify report was assigned
  • Check client account is active
  • Try refreshing the page
  • Log out and log back in

Automated Campaign Not Running

  • Check campaign status (active/paused)
  • Verify schedule settings
  • Check email delivery settings
  • Review campaign logs

Performance Optimization

  • Close unused browser tabs
  • Clear cache periodically
  • Use modern browsers (Chrome, Firefox, Safari, Edge)
  • Ensure stable internet connection
  • Allow sufficient time for AI analysis

Security & Privacy

Data Protection

  • All data encrypted in transit and at rest
  • Row Level Security prevents unauthorized access
  • Regular security audits performed
  • Compliant with data protection regulations

Password Security

  • Use strong, unique passwords
  • Change passwords regularly
  • Never share passwords via email
  • Enable two-factor authentication if available

Client Data Isolation

  • Each client sees only their assigned reports
  • No cross-client data visibility
  • Admin access is logged and monitored
  • Data deletion available on request
